### 商品详情：
```sql
CREATE TABLE `amazon_product_details_task`  
(  
    `id`          BIGINT       NOT NULL AUTO_INCREMENT,  
    `market_id`   VARCHAR(255) NOT NULL COMMENT '站点id',  
    `asin`        VARCHAR(255) NOT NULL COMMENT '商品asin',  
    `post_code`   VARCHAR(255) DEFAULT '' COMMENT '邮编',  
    `add_date`    VARCHAR(255) DEFAULT '1970-01-01' COMMENT '任务添加时间',  
    `state`       INT(11)      DEFAULT 0 COMMENT '任务状态',  
    PRIMARY KEY (`id`)
) ENGINE = InnoDB  
  AUTO_INCREMENT = 1  
  DEFAULT CHARSET = utf8;
```
#### 说明：
```python
id: 不用传
market_id: 支持的国家如上POST_CODE_JSON
asin: 必传
post_code: 可选，支持的国家邮编如上POST_CODE_JSON，不要传不支持的邮编，
			如果传空会随机使用当前站点已有cookie进行抓取
add_date: yyyy-mm-dd
state: 可选，传的话必须传0

```

### 榜单：
```sql
CREATE TABLE amazon_rank_list_product_task  
(  
    id          BIGINT AUTO_INCREMENT PRIMARY KEY,  
    market_id   varchar(255) NOT NULL COMMENT '站点id',  
    url         varchar(255) NOT NULL COMMENT '榜单链接',  
    url_type    VARCHAR(255) NOT NULL COMMENT '榜单类型（best或者new）',  
    category_id VARCHAR(255) NOT NULL COMMENT '类目id',  
    add_date    VARCHAR(255) DEFAULT '1970-01-01' COMMENT '任务添加时间',  
    state       INT(11)      DEFAULT 0 COMMENT '任务状态'  
) ENGINE = InnoDB  
  AUTO_INCREMENT = 1  
  DEFAULT CHARSET = utf8;
```
#### 说明：
```python
id: 不用传
market_id: 必传，支持的国家如上POST_CODE_JSON
url: 必传，例：https://www.amazon.com/gp/new-releases/electronics/18022313011
url_type: 必传，bestsellers/new-releases
category_id: 必传，例：18022313011
add_date: yyyy-mm-dd
state: 可选，传的话必须传0
```

### 搜索：
```python
CREATE TABLE `amazon_search_product_task`  
(  
    `id`        BIGINT AUTO_INCREMENT PRIMARY KEY,  
    `market_id` VARCHAR(255) NOT NULL COMMENT '站点id',  
    `post_code` VARCHAR(255) DEFAULT NULL COMMENT '邮编',  
    `keyword`   VARCHAR(255) NOT NULL COMMENT '搜索关键词',  
    `turn_page` VARCHAR(50)  NOT NULL COMMENT 'turn_page',  
    `add_date`  VARCHAR(255) DEFAULT '1970-01-01' COMMENT '任务添加时间',  
    `state`     INT(11)      DEFAULT 0 COMMENT '任务状态'  
) ENGINE = InnoDB  
  AUTO_INCREMENT = 1  
  DEFAULT CHARSET = utf8;
```
#### 说明：
```python
id: 不用传
market_id: 必传，支持的国家如上POST_CODE_JSON
post_code: 可选，支持的国家邮编如上POST_CODE_JSON，不要传不支持的邮编，
			如果传空会随机使用当前站点已有cookie进行抓取
keyword: 搜索关键词，必传
turn_page: 搜索页数，必传
add_date: yyyy-mm-dd
state: 可选，传的话必须传0
```
